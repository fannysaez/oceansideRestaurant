# 06 — Lancer le projet en local

Aucune dépendance à installer : le projet est en HTML/CSS statique avec Bootstrap chargé par CDN.

```bash
git clone https://github.com/fannysaez/oceansideRestaurant.git
cd oceansideRestaurant
```

Puis ouvrir simplement `index.html` dans le navigateur, ou lancer un serveur local (recommandé, notamment pour tester le comportement des chemins relatifs) :

```bash
# avec l'extension VS Code "Live Server", ou :
npx serve .
```
