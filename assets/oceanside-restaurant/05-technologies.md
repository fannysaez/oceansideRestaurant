# 05 — Technologies utilisées

| Technologie | Usage |
| --- | --- |
| **HTML5** | Structure sémantique (`header`, `nav`, `section`, `footer`) |
| **CSS3** | Variables custom, `linear-gradient`, `@media` mobile first |
| **Bootstrap 5.3.3** (CDN) | Grille, navbar responsive, formulaire, utilitaires (`d-none`, `d-md-block`, `gap-*`…) |
| **Google Fonts** | Cormorant Garamond / Inter |

Aucun build tool, aucun gestionnaire de paquets : le projet reste volontairement **statique**, chargé entièrement via CDN, pour rester simple à lire, cloner et déployer.
