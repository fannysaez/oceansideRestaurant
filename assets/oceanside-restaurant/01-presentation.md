# 01 — Présentation du projet

## Contexte

Projet réalisé dans le cadre de la formation **CDA — Concepteur Développeur d'Applications** à **Simplon**.

L'exercice consiste à **reproduire fidèlement en HTML/CSS/Bootstrap** une maquette Figma fournie en amont, intitulée « Oceanside Restaurant ».

---

## Objectif

Reproduire le site à l'identique en respectant :

- la **charte graphique** (couleurs, typographies, espacements) telle que définie dans la maquette Figma,
- la **structure des sections** (Hero, About, Menu, Reserve/CTA, Footer),
- une **mise en page responsive**, construite selon la méthode **mobile first** (les styles de base ciblent le mobile, puis sont enrichis via des `@media (min-width: ...)` pour tablette/desktop),
- l'utilisation du framework **Bootstrap 5** (grille, composants, utilitaires) plutôt que du CSS 100 % « from scratch ».

---

## Le site

Le site présente un restaurant fictif (« Oceanside Restaurant ») autour de 4 grandes sections :

| Section | Contenu |
| --- | --- |
| Hero | Accroche, titre, CTA « See menu » / « Place order », visuel décoratif |
| About | Présentation du lieu, statistiques (plats signature, couverts, cocktails) |
| Menu | 3 plats phares de « The Tide Menu » |
| Reserve / CTA | Formulaire de réservation + argumentaire |
